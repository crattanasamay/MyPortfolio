MY JOURNEY

Hello world, I graduated from the University of Illinois at Chicago with a degree in Computer Science. I decided to take a break due to COVID. Since then, I've gone through a major overhaul in what stack I wanted to work with. I started off working with node.js, express.js, and react as my way of creating web applications following the MVC design pattern. However, I wanted to change it up a bit and decided to learn C# in late August. Since then I've been working on my Anime Website and have been thinking of new ways of creating features as well as trying to optimally improve perfomance. Since learning C# I haven't had any lingering regrets as I've been having a lot of fun with coding in the Microsoft stack. I'm looking to for a place for me to learn and grow as .NET developer.
